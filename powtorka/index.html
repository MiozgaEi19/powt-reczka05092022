Źródła

Zdarzenia to czynności, które użytkownik wykonuje podczas odwiedzania naszej strony. Przykładowymi zdarzeniami mogą być np. przesunięcie kursora na obrazek, kliknięcie jakiegoś linka, wysłanie formularza, zaznaczenie jakiegoś obiektu, naciśnięcie klawisza itp. Można zaryzykować stwierdzenie, że żaden większy skrypt nie może obejść się bez zdarzeń. To właśnie one kreują tą pożądaną przez nas interaktywność... Większość zdarzeń wywoływana jest przez użytkownika. Użytkownik kliknie to, użytkownik zrobi tamto itp. Istnieją też zdarzenia, które nie są bezpośrednio spowodowane przez użytkownika - np. zdarzenie load, które zachodzi np. gdy załaduje się strona.

Click

Element odbiera zdarzenie kliknięcia, gdy przycisk urządzenia wskazującego (na przykład główny przycisk myszy myszy) zostanie naciśnięty i zwolniony, gdy wskaźnik znajduje się wewnątrz elementu. Jeśli przycisk zostanie naciśnięty na jednym elemencie, a wskaźnik zostanie przesunięty poza element przed zwolnieniem przycisku, zdarzenie zostanie wywołane na najbardziej konkretnym elemencie przodka, który zawierał oba elementy. Oto składnia oraz wykorzysatnie tego kodu w praktyce

HTML

<input type="button" class="on" id="click" value="on" onclick="zmiana_click()" />

JS

function zmiana_click(){ let x =document.getElementById("click") if (x.className=="on") { x.className="off" x.value = "off" } else{ x.className="on" x.value = "on" } }

Dblclick

Zdarzenie dblclick jest uruchamiane po dwukrotnym kliknięciu przycisku urządzenia wskazującego (takiego jak główny przycisk myszy); to znaczy, gdy szybko klikniesz dwa razy na pojedynczym elemencie w bardzo krótkim czasie. dblclick uruchamia się po dwóch zdarzeniach kliknięcia (a co za tym idzie, po dwóch parach zdarzeń mousedown i mouseup). Oto składnia oraz wykorzysatnie tego kodu w praktyce

HTML

<input type="button" class="on" id="dblclick" value="on" ondblclick="zmiana_dblclick()" />

JS

function zmiana_dblclick(){ let x =document.getElementById("dblclick") if (x.className=="on") { x.className="off" x.value = "off" } else{ x.className="on" x.value = "on" } }

Mousedown

Zdarzenie mousedown jest uruchamiane na elemencie, gdy przycisk urządzenia wskazującego zostanie naciśnięty, gdy wskaźnik znajduje się wewnątrz elementu. Uwaga: różni się to od zdarzenia kliknięcia tym, że kliknięcie jest uruchamiane po wykonaniu pełnej akcji kliknięcia; oznacza to, że przycisk myszy jest wciskany i zwalniany, gdy wskaźnik pozostaje wewnątrz tego samego elementu. Mousedown jest uruchamiany w momencie pierwszego naciśnięcia przycisku. Oto składnia oraz wykorzysatnie tego kodu w praktyce

HTML

<input type="button" class="on" id="mousedown" value="on" onmousedown="zmiana_mousedown()" />

JS

function zmiana_mousedown(){ let x =document.getElementById("mousedown") if (x.className=="on") { x.className="off" x.value = "off" } else{ x.className="on" x.value = "on" } }

mouseout

Zdarzenie mouseout jest uruchamiane na elemencie, gdy urządzenie wskazujące (zwykle mysz) jest używane do przesuwania kursora, tak aby nie był już zawarty w elemencie lub jednym z jego elementów podrzędnych. Mouseout jest również dostarczany do elementu, jeśli kursor wejdzie w element potomny, ponieważ element potomny zasłania widoczny obszar elementu. Oto składnia oraz wykorzysatnie tego kodu w praktyce

HTML

<input type="button" class="on" id="mouseout" value="on" onmouseout="zmiana_mouseout()" />

JS

function zmiana_mouseout(){ let x =document.getElementById("mouseout") if (x.className=="on") { x.className="off" x.value = "off" } else{ x.className="on" x.value = "on" } }

change

Zdarzenie change jest wywoływane dla elementów <input>, <select> i <textarea>, gdy użytkownik zatwierdzi zmianę wartości elementu. W przeciwieństwie do zdarzenia input, zdarzenie change niekoniecznie jest uruchamiane dla każdej zmiany wartości elementu. Oto składnia oraz wykorzysatnie tego kodu w praktyce

HTML

<input type="text" id="change" onchange="zmiana_change()">

JS

function zmiana_change() { document.getElementById("change").style.backgroundColor = "blue";}
